# Рядовой Авангард — Telegram Bot

Серверная часть: бот, который открывает Mini App и (в будущем) хранит прогресс
игроков, шлёт напоминания «пора брить Олега» и ведёт статистику.

> Каркас. Логика будет наполняться по мере развития проекта.

## Структура

```
bot/
  src/
    index.ts    — точка входа, регистрация команд, запуск
    config.ts   — конфиг из переменных окружения
    users.ts    — работа с пользователями и их прогрессом (заглушка)
  .env.example  — пример переменных окружения
  package.json
```

## Запуск (когда будет реализовано)

```bash
cd bot
cp .env.example .env      # вписать BOT_TOKEN и WEBAPP_URL
npm install
npm run dev
```

## Планы

- `/start` → кнопка, открывающая Mini App (`WEBAPP_URL`).
- Хранение прогресса (день щетины, последнее бритьё) на сервере, а не только в
  localStorage — чтобы прогресс не терялся между устройствами.
- Напоминания через бота, когда щетина отросла (день 7).
- Валидация `initData` Telegram для безопасной идентификации пользователя.
